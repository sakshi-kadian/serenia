# Serenia 

Welcome to **Serenia**, a digital sanctuary designed to help you understand, explore and improve your emotional well-being. Built with empathy at its core, Serenia combines the power of generative AI with therapeutic journaling and mood tracking to bridge the gap between emotional support and data-driven self-awareness.

---

## Built With

- **Frontend**: Next.js, Tailwind CSS, Framer Motion
- **Backend**: Python FastAPI, Uvicorn, Pydantic
- **Database**: SQLAlchemy, PostgreSQL (Supabase)
- **NLP & ML**: PyTorch, spaCy, NLTK, Hugging Face Transformers (BERT, RoBERTa)
- **AI Core**: Google Gemini 2.5 Flash (Generative Intelligence)
- **Authentication**: Clerk

---

## Getting Started

### 1. Start the Backend Server
```bash
cd backend
# Activate virtual environment
.\venv\Scripts\activate
# Run the server
python main.py
```

### 2. Start the Frontend Application
```bash
cd frontend
# Run the development server
npm run dev
```
